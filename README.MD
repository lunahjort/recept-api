# Recept-API

Ett REST API byggt för att öva på backend-utveckling med Node.js och Express.

## Endpoints
- `GET /recept` – hämta alla recept
- `GET /recept/:id` – hämta ett specifikt recept
- `POST /recept` – skapa ett nytt recept
- `PUT /recept/:id` – uppdatera ett recept
- `DELETE /recept/:id` – ta bort ett recept

## Tekniker
- Node.js
- Express
- MongoDB Atlas
- Mongoose
